# World Population Data Visualization Project

## Overview
This project analyzes world population data based on gender using Python data visualization libraries.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Features
- Gender-wise population analysis
- Bar charts
- Pie charts
- Histogram visualization
- Data cleaning and preprocessing

## Dataset
World population dataset containing male and female population statistics.

## Learning Outcomes
- Data visualization techniques
- Exploratory Data Analysis (EDA)
- Plotting graphs using Matplotlib and Seaborn
- Handling datasets using Pandas

## Output
Visual representation of gender distribution in world population.
